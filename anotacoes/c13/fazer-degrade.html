Para fazer um degradê é preciso usar a atribuição:

background-image: linear-gradient(direção, cor1, cor2);

Um exemplo real é o usado neste site que foi:

background-image: linear-gradient(to right, #3400DB, #0010DB);

Para colocar o degradê em direções diferentes bastar usar:

  • direção: to right (para direita)
  • direção: to left (para esquerda)
  • direção: to top (para cima)
  • direção: to bottom (para baixo)

Caso o site fique zebrado ao colocar "to top" e "to bottom", use este código que eixa o degradê fixo no fundo.

background-attachment: fixed;
