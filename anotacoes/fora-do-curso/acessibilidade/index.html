Ultilizando [aria-label]

Aria label é ultilizada para rotular um elemento que não contém texto nenhum para ser descrito como por exemplo um ícone de menu. É principalmente usado para Acessibilidade

Aria labeledby

Ultilizado para referenciar um título adequando para o elemento através de 1 ou mais IDs apenas.

Aria describedby

Ultilizado para referenciar uma descrição adequada para o elemento através de 1 ou mais IDs apenas.
