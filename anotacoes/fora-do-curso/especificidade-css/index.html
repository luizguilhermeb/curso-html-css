Ordem de precedência nas CSS

  1. ID
  2. Classes, Atributos de Classe, Pseudo-classes
  3. Elementos, Pseudo Elementos

Exemplos

Entre um ID e uma classe, a classe prevalece.

Entre duas classes iguais, a que estiver por último prevalece por padrão, porém você pode aumentar o nível de especificidade do seletor, fazendo com que ele prevaleça.
