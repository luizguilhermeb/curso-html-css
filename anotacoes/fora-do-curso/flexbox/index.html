Flexbox

Praticamente o Flexbox se resume a um elemento pai que contém filhos que vão se adaptar flexivelmente a ele.

Sempre há um eixo principal (main axis) que por padrão começa da esquerda e vai para a direita e um eixo perpendicular (cross axis).

Código usado para o container acima:

.container {
        background-color: antiquewhite;
        padding: 10px;
    
        display: flex;
        flex-direction: row; /*Direção do Flex*/
        flex-wrap: wrap; /* Quebra de linha */
        flex-flow: row wrap; /* Junção de row e wrap */
    
        justify-content: center; /* Justificação do Conteúdo Principal*/
        
        align-items: center; /* Justificação do Conteúdo Perpendicular (Por Linha)*/ 
        align-content: center; /* Justificação do conteúdo Perpendicular (Decidir se os elementos vão ficar espremidos, espalhados ou para algum lado) */
    
        gap: 10px 20px; /* espaçamento  interno de cada item */
    }
    
    .bloco {
        background-color: aquamarine;
        border-radius: 10px;
        width: 100px;
        height: 100px;
        flex-grow: 1; /*Seu valor padrão é 0, colocando 1 torna possível que os itens cresçam, aumentando esse valor, torna possível que uns cresçam mais sobre os outros*/
        flex-shrink: 1; /* Seu valor padrão é 1, ou seja, já tem a habilidade de diminuir, aplicando o valor 2 essa habilidade aumenta, podendo diminuir mais do  que os outros. */
        flex-basis: auto; /* Defina a largura base do elemento caso o main axis seja row, ou a altura base do elemento caso o main axis seja column; além disso colocando auto como padrão, o conteúdo dentro será usado no flex-grow */
        flex: 1 1 auto; /* Shorthand para flex-grow/flex-shrink/flex-basis */
    }
    
    .bloco:nth-child(even) {
        background-color: chartreuse;
    }
    
    .bloco:nth-child(23) {
        flex-grow: 8; /*Seu valor padrão é 0, colocando 1 torna possível que os itens cresçam, aumentando esse valor, torna possível que uns cresçam mais sobre os outros*/
        align-self: flex-end; /* o atributo align-content se sobrepõe ao align-self que serve para mudar individualmente a posição no eixo perpendicular. */
    }
    
    
    .bloco:last-child {
        order: -1;
        background-color: green;
    }
