Seletores CSS (elementos-filho)

O exemplo abaixo diz que todo p e h1 dentro de toda section terá color: red;: 

section p,
section h1 {
    color: red;
}

Também é possível usar "*" no lugar para dizer "todos os elementos".

Adjacent Sibling selector (+)

Este seletor serve para dizer que o primeiro elemento que vier depois de outro receberá certo estilo, como aqui:

.pai3  p+h2 /*Todo primeiro h2 que vier depois do p será green*/ {
    color: green;
}

.pai3 h2+p /*o mesmo serve para o p*/ {
    color: red;
}

General sibling selectors (~)

Este serve para dizer que todos os elementos de tal tipo que vierem depois de tal elemento, receberá certo estilo.

.pai4 h2~p{
    color: purple;
}
