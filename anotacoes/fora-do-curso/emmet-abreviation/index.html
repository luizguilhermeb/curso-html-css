Emmet Abreviation

HTML

Abreviar div com classe e id: div#identificador.uma_classe.outra_classe

Abreviar div com classe e id, e multiplaca-la: div.uma_classe.outra_classe*4

Abreviar div com classe e id, e multiplaca-la, enumerando os id: div#identificador-$.uma_classe.outra_classe*4

Abreviar elementos dentro de elementos: ul>li{$}*4

Abreviar elementos dentro de elementos por grupos: ul>(li>a{$})*4

Abreviar elementos especificando atributos: div>a[href="https://youtube.com"]

Abreviar elementos com filhos diferentes: div>h1+p*4+div*2

CSS

Nas CSS é mais simples, bastar usar expressões como "w10px" para width: 10px; e "h20px" para height: 20px; (desconsidere as aspas)
