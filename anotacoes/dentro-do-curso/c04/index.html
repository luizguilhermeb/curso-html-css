Definições Importantes

<!DOCTYPE html> tem a função de dizer para o navegador renderizar o site na versão mais atual do html, para que ele não rode no quirks mode, um modo feito para rodar sites antigos que ainda não seguiam o padrão atual do html.

<meta http-equiv="X-UA-Compatible" content="IE=edge"> tem a função de renderizar o site da melhor forma no navegador Internet Explorer.

<meta name="viewport" content="width=device-width, initial-scale=1.0"> tem a função de deixar o site com uma escala proporcial a tela dos celulares e outros dispositivos.
