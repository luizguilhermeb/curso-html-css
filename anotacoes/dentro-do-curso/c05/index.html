HTML ENTITIES

html entities são códigos usados dentro do html para inserir caracteres especiais, para consulta-los basta pesquisar por "html entities".

EMOJIS

Para colocar emojis no site, pesquise por emojipedia.org, escolha o emoji, e copie seu codepoint, por exemplo U+1F60E.

Após, troque o U+ por &#x. ex: (& # x 1 F 6 0 E ;) não monoespace como no exemplo.
