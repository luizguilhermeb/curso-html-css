Parâmetro target

O parâmetro target suporta os seguintes valores:

‣ _blank vai abrir um link em uma nova janela em branco(Youtube)
‣ _self vai abrir o link na janela ou frame atual (padrão)(Youtube)
‣ _top vai desfazer todos os frames e abrir o destino no navegador completo(Youtube)
‣ _parent similar ao uso do _top em uma referência à janela mãe(Youtube)
‣ nome-do-frame caso esteja usando frames, indicar o nome da janela a abrir.

Indicar função do link

‣next indica que o link é para a próxima parte do documento atual Página 2
‣prev indica que o link é para a parte anterior do documento atual Página 2
‣author indica que é um link para o site do autor do artigo atualPágina 2
‣external indica que é um link para outro site que não faz parte do site atualPágina 2
‣nofollow indica que é um link para um site não endossado, como um link pago Página 2
