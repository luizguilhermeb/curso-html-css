Media Queries

São configurações que ajudam o site a ser responsivo

Media Types

São configurações de media que aplicam estilos específicos para tipos de media

<link rel="stylesheet" href="estilos/tela.css" media="screen">
<link rel="stylesheet" href="estilos/impressora.css" media="print">

Media Features

São configurações de media que aplicam estilos específicos para tipos de media com certas características

Ex: Verificar se um aparelho está em modo portrait(retrato) ou landscape(paisagem)

<link rel="stylesheet" href="estilos/retrato.css" media="screen and (orientation: portrait)">

Ou até no próprio CSS

@media screen and (orientation: portrait) {
    body, html {
        background: #233EFF url(../imagens/cev-portrait.jpg) no-repeat center bottom / contain;
    }
}

Typical Device Breakpoints

São pontos específicos de media-query, em que o site muda certos estilos para certos dispositivos de acordo com sua resolução.

Menu hamburguer

Para fazer um menu responsivo e profissional é necessário usar Javascript. Como no exemplo abaixo:

HTML

<span class="material-symbols-outlined" id="icone-menu" onclick="clickMenu()">menu</span>
    <nav id="itensNav">
        <ul>
            <li><a href="#">Sobre</a></li>
            <li><a href="#">Fale Conosco</a></li>
            <li><a href="#">Notícias</a></li>
            <li><a href="#">Orçamento</a></li>
            <li><a href="#">Avalie-nos</a></li>
        </ul>
    </nav>

Javascript

<script>
    function clickMenu() {
        if (itensNav.style.display == "none") {
            itensNav.style.display = "block";
        } else {
            itensNav.style.display = "none";
        }
    }
</script>
