Adições importantes

Overflow

Propriedade CSS usada para estilizar elementos que passam dos limites de seu elemento pai podendo oculta-los, criar uma barra de rolagem e etc...

Float

DIV 1
DIV 2

Uma Propriedade CSS usada para alinhar elementos em certo sentido dentro de um conteiner, isso os tira de seu contexto-primário no html.

Clear

A propriedade clear no CSS é usada para posicionar um elemento abaixo dos elementos flutuantes, sendo configurado como clear: left, clear: right ou clear: both. É útil para evitar que elementos subsequentes fiquem presos aos elementos flutuantes.

Calc

Propriedade das CSS que calcula medidas como px, vh e vw nas css. EX: calc(100vh - 100px)
