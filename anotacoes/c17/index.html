Video do Youtube Responsivo

Para fazer um <iframe> responsivo é necessário entender os seguintes conceitos:

  • position: relative e absolute
  • Uso de porcentagem em atributos de caixa; ex - padding: 50%;

Position

Um elemento pode ter position absolute ou relative, sendo que seu elemento padrão é static.

Caixa2

Quando se coloca absolute ou relative, 4 novos atributos são desbloqueados para uso, (top, bottom, right, left).

Um elemento com position relative tem sua posição em relacão a ele mesmo.

Já um elemento absolute tem sua posição em relação ao primeiro elemento pai com atributo absolute, relative, fixed e sticky.

A caixa no topo da página tem o seguinte código:

#caixa-absolute {
    background-color: aqua;
    width: 400px;
    height: 100px;
    position: absolute;
    top: 30px;
    right: 30px;
}

Uso de porcentagem

Para entender o uso de porcentagem:

  • Verifique a documentação para cada propriedade CSS que usa valores em porcentagem.
  • Compreenda o contexto em que a porcentagem está sendo usada (por exemplo, em relação ao elemento pai ou ao próprio elemento).
  • Conheça as limitações de cada propriedade CSS em relação ao uso de valores em porcentagem.
